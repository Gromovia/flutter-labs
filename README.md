# Проект: Приложение "Список дел" (To-Do List)

## Краткое описание
Это простое мобильное или веб-приложение для управления личными задачами. Оно позволяет пользователям создавать, просматривать, редактировать и удалять задачи, а также отмечать их как выполненные. Цель проекта — освоить базовые CRUD-операции и взаимодействие с пользовательским интерфейсом.

---

## Основные функции

1. **Создание задач:**
    * Добавление названия задачи (обязательно).
    * Добавление описания задачи (опционально).
    * Установка срока выполнения (опционально, с выбором даты).
2. **Просмотр списка задач:**
    * Отображение всех активных и завершенных задач.
    * Фильтрация по статусу (активные / завершенные).
    * (На будущее) Сортировка по дате создания / сроку выполнения.
3. **Изменение/редактирование задач:**
    * Изменение названия, описания и срока выполнения существующей задачи.
    * Изменение статуса задачи (отметить как выполненную / снять отметку).
4. **Удаление задач:**
    * Удаление конкретной задачи.

## Структура экранов

### 1. Главный экран: "Список задач"
* **Заголовок:** "МОИ ЗАДАЧИ"
* **Элементы:**
    * **Вкладки/Фильтры:** "Активные" и "Завершенные" для переключения между списками задач.
    * **Список задач:** Каждая задача отображается в отдельном блоке, содержащем:
        * Чекбокс для быстрого изменения статуса.
        * Название задачи.
        * Краткое описание или срок выполнения (если заданы).
        * Стрелка ">" для перехода к экрану редактирования.
    * **Плавающая кнопка "+" (FAB):** Для быстрого добавления новой задачи, расположена внизу экрана.

### 2. Экран "Создание/Редактирование задач"
* **Заголовок:** "НОВАЯ ЗАДАЧА" (при создании) / "РЕДАКТИРОВАТЬ ЗАДАЧУ" (при редактировании).
* **Элементы:**
    * **Кнопка "Название" (<):** Возврат на главный экран (слева вверху).
    * **Кнопка "Сохранить" (✓):** Сохранение изменений или новой задачи (справа вверху).
    * **Поле ввода для названия задачи:** Обязательное текстовое поле.
    * **Поле ввода для описания задачи:** Многострочное текстовое поле (опционально).
    * **Поле/Кнопка "Срок выполнения":** Для выбора даты (опционально).
    * **Кнопка "Удалить задачу":** Появляется только в режиме редактирования задачи.

---

## Навигация

1. **Главный экран ("Список задач")**
    * Нажатие на **"+"** -> Переход на экран "Создание/Редактирование задач" (режим создания).
    * Нажатие на **блок задачи** (или ">") -> Переход на экран "Создание/Редактирование задач" (режим редактирования).
    * Нажатие на **вкладки "Активные"/"Завершенные"** -> Переключение между списками задач.
    * Нажатие на **чекбокс задачи** -> Изменение статуса задачи (без перехода на другой экран).

2. **Экран "Создание/Редактирование задач"**
    * Нажатие на **"< Название"** -> Возврат на главный экран без сохранения изменений.
    * Нажатие на **"Сохранить" (✓)** -> Сохранение задачи и возврат на главный экран.
    * Нажатие на **"Удалить задачу"** (в режиме редактирования) -> Диалог подтверждения -> Удаление задачи и возврат на главный экран.
